🔥 Configurar Firebase

Para sincronizar entre dispositivos necesitas crear un proyecto Firebase gratuito y pegar tu configuración aquí. Solo se hace una vez.

Paso 1 — Ve a console.firebase.google.com e inicia sesión con tu Gmail.
Paso 2 — Haz clic en "Agregar proyecto", ponle un nombre (ej: sabornatura) y créalo (puedes desactivar Analytics).
Paso 3 — Dentro del proyecto, haz clic en el ícono </> Web, registra la app y copia el objeto firebaseConfig.
Paso 4 — Ve a Firestore Database → Crear base de datos en modo producción. Luego en la pestaña Reglas pega esto y publica:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
